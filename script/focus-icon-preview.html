Instructions for Saving Icons

  1. Right-click on each icon and select "Save Image As..." to save it to your computer.
  2. Save each icon with the appropriate filename (16.png, 32.png, 48.png, 96.png, 128.png).
  3. Place the saved PNG files in your extension's "public/icon/" directory.
  4. Toggle between light and dark mode to ensure the icons look good in both themes.
